Alpha Tester Information

Welcome to the Bang! Howdy Alpha Test! This here page is chock full of information on how to get started and have yerself some Wild West fun.

Read this First
This is an Alpha Test which means a few things:

  • Please don't share any of the information we provide to you on any public forums. No postin' screenshots, no postin' these URLs. If you want to invite a few friends to come and play, that's great but otherwise keep it under yer hat. We'd like to slowly invite more people to the test to avoid being overwhelmed.
  • The game ain't done yet. Some of the boards aren't very purdy and exist so that we can test out gameplay ideas. Everything is subject to change. We'll try to never have to delete anything or mess up yer accounts, but this here's the Wild West and if push comes to shove, we have to do everything we can to make the game great.
  • We really appreciate your help in testin' out the game and we'll definitely have something special for ya when all is said and done and we're ready to go live. So thanks in advance fer being a great tester.

Create an Account
If you already have an account for Puzzle Pirates or Game Gardens, you can use that same account on Bang! Howdy. Otherwise, use the create account page to make yourself a new account.

Download and Install the client
Download whichever of these here installers is appropriate for your platform and run it:

  • Windows installer
    (The installer will automatically download and install Java for you if you don't already have the necessary version)
  • Linux installer
    (You must have Java 1.5 installed on your Linux machine)
  • MacOS X installer
    (You must have Java 1.5 installed on your Mac and must be running MacOS X 10.4.2 or later)

Read the Alpha Tester Forums
Check the forums for information on:

  • Testing schedule
  • Bugs and problems
  • Anything else you might want to chat about

Let Us Know if Ya Have Problems
If you have a problem in the game, you can press F2 to bring up the bug report dialog and submit a bug. This is useful because it will send us a debug log from your client.

If you are having trouble gettin' the game installed or running at all, post to the technical issues forum and tell us what kind of computer you got, what operating system and any other details you think might help us to figger out what's goin' awry.

If ya can't even get into the forums, then send me an email and we'll try to get that sorted out. Everyone in the Alpha Test should already have an account with Three Rings for Puzzle Pirates or Game Gardens, and that account should work for Bang! Howdy too (the forums and the game). If it ain't workin' or you don't have one, send me email at mdb@threerings.net and we'll sort you out.
